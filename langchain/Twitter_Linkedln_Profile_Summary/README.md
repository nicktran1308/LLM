
# Twitter/Linkedln_profile_scrape

Follow Eden Marco (Github: emarco177) instruction from Langchain course to create a Linkedln & Twitter scraping profile.

## Technologies used:

Langchain
Serp API
Tweepy
Proxycurl
Apache Tomcat

## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`PYTHONPATH=/{YOUR_PATH_TO_PROJECT}/ice_breaker`

`OPENAI_API_KEY`

`PROXYCURL_API_KEY`

`SERPAPI_API_KEY`

`TWITTER_API_KEY`

`TWITTER_API_SECRET`

`TWITTER_ACCESS_TOKEN`

`TWITTER_ACCESS_SECRET`
## Run Locally

Clone the project

```bash
  git clone "URL"
```

Go to the project directory

```bash
  cd Twitter_Linkedln_Profile_Summary
```

Install dependencies

```bash
  pipenv install
```

Start the flask server

```bash
  pipenv run app.py
```


## Running Tests

To run tests, run the following command

```bash
  pipenv run pytest .
```
